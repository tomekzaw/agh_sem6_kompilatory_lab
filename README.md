# Interpreter

## Usage
```bash
python main.py fib.m
```
Linux:
```bash
cat fib.m | python main.py
```
Windows:
```bash
type fib.m | python main.py
```

```bash
$ python main.py
   1 |  n = 3;
   2 |  A = eye(n);
   3 |  print A;
   4 |  ^Z
[[1. 0. 0.]
 [0. 1. 0.]
 [0. 0. 1.]]
```
